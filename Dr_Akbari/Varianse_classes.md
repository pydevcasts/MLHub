## ✅ گام 1: نرمال‌سازی داده‌ها در بازه \[–1, 1]

فرمول نرمال‌سازی:

```math
x' = \left(\frac{x - x_{\min}}{x_{\max} - x_{\min}} \times 2\right) - 1
```

### 🔍 حداقل و حداکثر برای هر ویژگی

| ویژگی | min | max |
| :---: | :-: | :-: |
|   F1  |  1  |  8  |
|   F2  |  2  |  9  |
|   F3  |  2  |  8  |
|   F4  |  5  |  9  |

### 🔢 نرمال‌سازی داده تست

* **F1:**

  ```
  ((1.5 – 1) / 7) × 2 – 1 = –6/7
  ```
* **F2:**

  ```
  ((7 – 2) / 7) × 2 – 1 = 1
  ```
* **F3:**

  ```
  ((2 – 2) / 6) × 2 – 1 = –1
  ```
* **F4:**

  ```
  ((7 – 5) / 4) × 2 – 1 = 0
  ```

✅ **داده تست نرمال‌شده:**

```
test' = (–6/7, 1, –1, 0)
```
